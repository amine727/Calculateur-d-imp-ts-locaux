#  Calculateur d'impôts locaux

##  Description

Application Android permettant de calculer les impôts locaux selon :

* la surface du logement
* le nombre de pièces
* la présence d'une piscine

---

##  Calcul

* Impôt de base = surface × 2
* Impôt supplémentaire = (pièces × 50) + (100 si piscine)
* Impôt total = base + supplémentaire
